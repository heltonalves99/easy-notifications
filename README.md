# easy-notifications [alpha]
A python app to manage subscriptions and send push notifications to mobile devices. 


## Cloning repo
```
	$ git clone git@github.com:raelmax/easy-notifications.git
	$ cd easy-notifications
```

## Sync database
```
    make sync
```

## Run app
```
	python run.py
```

## Run tests
```
	make test
```
